# 🚀 Começar Agora - Guia Rápido (5 Minutos)

## ⏱️ Tempo Estimado: 5 minutos

---

## Passo 1: Abrir a Aba Produtos

```
1. Abra o VALEI-ME (seu sistema)
2. Olhe o menu à esquerda (barra preta/cinza)
3. Clique em "Produtos" (ícone de caixa 📦)
```

**Resultado**: Você vê uma página com título "📦 Catálogo de Produtos"

---

## Passo 2: Criar Seu Primeiro Produto

```
1. Clique no botão azul "+ Novo Produto"
   (está no topo, abaixo do título)

2. Um formulário aparece:
   ┌─────────────────────────────────┐
   │ Nome: [_________________]        │
   │ Categoria: [Geral ▼]             │
   │ Preço de Venda: [____]           │
   │ Estoque Mínimo: [____]           │
   │ Descrição: [___________]         │
   │ [Salvar]  [Cancelar]             │
   └─────────────────────────────────┘

3. Preencha os campos:
   - Nome: Escreva o nome do seu produto
     Exemplo: "Bolo de Chocolate"
   
   - Categoria: Deixe "Geral" ou escolha outra
     (ou crie uma nova digitando)
   
   - Preço: Quanto você vai cobrar?
     Exemplo: "45,00" ou "45"
   
   - Estoque Mínimo: Deixe em branco (opcional)
   
   - Descrição: Deixe em branco (opcional)

4. Clique "Salvar"
```

**Resultado**: Seu produto aparece na grade com:
- Nome
- Preço em verde (R$ 45,00)
- Status "Ativo" (badge verde)
- Botões: Editar, Ativar/Desativar, Vincular, Deletar

---

## Passo 3: Vincular à Ficha Técnica (Opcional mas Recomendado)

Se você quer que o custo seja calculado automaticamente:

```
1. No seu produto, clique "Vincular Ficha"
   (botão com ícone de corrente/link)

2. Um modal abre mostrando:
   ┌─────────────────────────┐
   │ FICHAS TÉCNICAS         │
   │ ────────────────────    │
   │ □ Bolo de Chocolate     │
   │   R$ 15,00             │
   │                        │
   │ □ Brigadeiro           │
   │   R$ 4,00              │
   │                        │
   │ [Fechar]               │
   └─────────────────────────┘

3. Clique na Ficha que corresponde ao seu produto
   Exemplo: Clique "Bolo de Chocolate"

4. Automaticamente:
   - Custo aparece: R$ 15,00
   - Margem calcula: 67%
   - Modal fecha
   - Seu produto atualiza

5. Na grade, veja:
   - Custo: R$ 15,00 (puxado da Ficha)
   - Margem: 67% (calculado automático)
```

**Resultado**: Produto vinculado à receita, custo sincronizado!

---

## Passo 4: Verificar no Dashboard

Para ver se tudo sincronizou:

```
1. No menu lateral, clique "Início"
   (ou "/" - primeira opção do menu)

2. Procure o card azul com título:
   "📦 Produtos Ativos"

3. Deverá mostrar: 1 (seu novo produto)

4. Se desativar um produto:
   - Volte para "Produtos"
   - Clique "Desativar"
   - Volte para "Início"
   - "Produtos Inativos" mudará para: 1

(Tudo sincroniza em TEMPO REAL!)
```

---

## Pronto! 🎉

Você completou:
- ✅ Criar um produto
- ✅ Vincular a uma ficha técnica
- ✅ Ver sincronização no dashboard

Agora você pode:
1. **Criar mais produtos** (repita Passo 2)
2. **Editar preços** (Clique "Editar" no produto)
3. **Desativar/Ativar** (Clique botão de interruptor)
4. **Ver no PDV** (abra Vendas → Veja produto na lista)

## 💡 Dicas Rápidas

### Pesquisar Produto
```
Se tiver muitos produtos:
1. No topo da página Produtos
2. Clique na caixa "Pesquisar por nome..."
3. Digite o nome
4. Lista filtra em tempo real
```

### Editar Produto
```
1. Clique "Editar" (lápis) no produto
2. Formulário abre com dados preenchidos
3. Altere o que quiser
4. Clique "Salvar Alterações"
```

### Desativar Temporariamente
```
Faltou ingrediente? Não quer vender por enquanto?
1. Clique "Desativar" (interruptor) no produto
2. Produto some da lista de vendas
3. Para restaurar: Clique "Ativar"
```

### Deletar (Cuidado!)
```
Não vai mais fazer esse produto?
1. Clique "Deletar" (lixeira) no produto
2. Confirme: "Tem certeza?"
3. Produto é removido permanentemente

⚠️ Dica: Use "Desativar" se pode voltar a fazer!
```

## 📞 Problemas Comuns

### "Produto não aparece no PDV"
✅ Solução: Aba Produtos → Clique "Ativar" → Abra PDV novamente

### "Custo ficou errado"
✅ Solução: Clique "Atualizar Custo" se mudou a Ficha Técnica

### "Como mudo o preço?"
✅ Solução: Clique "Editar" → Altere "Preço de Venda" → Salvar

### "Como vendo via PDV?"
✅ Solução: Menu → "Vendas" → Escolha seu produto → Venda!
